**Пример реализации бэкенда для сайта жилоо комплекса (Symfony 5.x)**

**Требования**

```
PHP 7.4
MySQL 5.7
Composer 2.x
```
## Установка
```$ composer install```

```$ npm install```

```$ yarn install```

```$ yarn encore dev```
## Команды

1. Очистка загруженных (временных) файлов
Команду нужно добавить в крон, иначе место однажды закончится.
```
$ ajax_uploader:clean
```
